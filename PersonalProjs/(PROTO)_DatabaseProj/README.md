HELLO, welcome to the readme

This command line terminal is meant to be ran with the sqlite3 library
compiled using:

g++ <cpp/etc files> <flag (usually -o)> -lsqlite3 //sqlite library

SQLITE IS NEEDED ON BASH IN ORDER TO RUN AND INCLUDE THE NECESSARY HEADERS. 

If you either don't know if you have the library or know you dont have it, utilize the installation script with the following:

chmod +x install.sh
./install.sh

This will require input for the users credentials.

If you do or finished installing them on your linux machine, you can just execute the program with the following:

chmod +x run.sh
./run.sh

NOTE: not everything is bulletproof so if something breaks let me know and Ill see what went wrong. Thank you, hopefully you enjoy the program at least a little :)
